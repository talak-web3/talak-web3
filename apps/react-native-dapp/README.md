# react-native-dapp

Example [Expo](https://expo.dev/) / React Native app using **`@talak-web3/core`** and **`@talak-web3/hooks`** with React Navigation.

## Prerequisites

- Node.js 20+
- pnpm
- For device testing: Expo Go or a local Android/iOS toolchain

## Setup

From the monorepo root:

```bash
pnpm install
pnpm --filter react-native-dapp dev
```

This runs `expo start` so you can open the app in a simulator or Expo Go.

## Scripts

| Script | Description |
|--------|-------------|
| `dev` | `expo start` |
| `build` | Placeholder build step (see `package.json`) |
| `typecheck` | TypeScript check |
