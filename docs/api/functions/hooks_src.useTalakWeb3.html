    • useTalakWeb3(): TalakWeb3Instance

      Escape-hatch hook for advanced use cases.

      Returns the full SDK instance for direct access to plugins, middleware, and anything not covered by the higher-level hooks.

      Returns TalakWeb3Instance

      Example 

      function PluginList() {
        const instance = useTalakWeb3();
        return (
          <ul>
            {Array.from(instance.context.plugins.values()).map(p => (
              <li key={p.name}>{p.name} v{p.version}</li>
            ))}
          </ul>
        );
      }
